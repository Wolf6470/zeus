# ⚡️ Zeus Panel Auto-Deployer

این پروژه یک پنل زیبا و سبک است که بر بستر **ورکر کلودفلر** اجرا می‌شود و به شما امکان مدیریت پروتکل‌ها و کاربران را می‌دهد. هدف این مخزن، تسهیل فرآیند مستقر‌سازی خودکار این پنل است.

## 🚀 ویژگی‌های کلیدی
* **مدیریت کاربران:** قابلیت اضافه کردن کاربر جدید و نظارت بر وضعیت آن‌ها.
* **سیستم هوشمند وضعیت اشتراک:** نمایش وضعیت فعال، مسدود شده، اتمام ترافیک مجاز یا منقضی شدن زمان اشتراک کاربران.
* **نصب و راه‌اندازی تمام خودکار:** مستقر‌سازی سریع و آسان روی حساب کلودفلر شما تنها با چند کلیک در محیط وب، بدون نیاز به دانش فنی یا اجرای هیچ‌گونه کدی.
* **عملکرد بهینه:** استفاده از معماری پردازش ابری کلودفلر برای سرعت بالا و پایداری فوق‌العاده.

## 🛠 نحوه استفاده

برای ساخت و راه‌اندازی پنل اختصاصی خود، نیازی به اجرای اسکریپت ندارید. فقط کافیست مراحل زیر را دنبال کنید:

۱. ابتدا وارد سایت دیپلوی خودکار زیر شوید:

### 🌐 **[https://zeus-panel.ir-netlify.workers.dev/](https://zeus-panel.ir-netlify.workers.dev/)**

۲. در سایت، روی گزینه **"دریافت توکن"** کلیک کنید. (این کار شما را به کلودفلر هدایت می‌کند تا توکنی با دسترسی‌های دقیق و لازم برای ساخت پنل ایجاد کنید).

۳. توکن دریافت شده را کپی کرده، در کادر مربوطه در سایت قرار دهید و روی **"ساخت پنل"** کلیک کنید. 

۴. منتظر بمانید تا فرآیند ساخت دیتابیس، ساب‌دامین و آپلود ورکر به صورت خودکار انجام شود و لینک نهایی پنل به شما نمایش داده شود.
